# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1, as its code of conduct. The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

In short: we want SAMBA to be a welcoming, harassment-free community for everyone.
Be respectful and constructive in issues, pull requests, and discussions.

## Reporting

To report a concern, contact the maintainers privately via GitHub — open a private
report under the repository's **Security** tab, or reach the repository owner
([@CameronBrooks11](https://github.com/CameronBrooks11)). Reports are handled
confidentially.
